path.join() and path.resolve()

path is one of the many modules present inside the node.js librarypathmodule provides us with the utilities for working with file and directory paths.

  • path utilities can be used inside any node project by just importing the path module using the require function as follows.

path.join()

The path.join() method joins all given path segments together using the platform-specific separator as a delimiter (based on the OS the node is running on.), then normalizes the resulting path(joins the path and then evaluates if any parent(..) directory or any symbols representing the path navigation in command line.).

syntax

  • The path.join() method returns a string if the execution is completed successfully and a TypeError is thrown if any of the path segments is not a string.

path.resolve()

The path.resolve() method resolves a sequence of paths or path segments into an absolute path.(i.e. converts a set of segments into the absolute path.)

  • The given sequence of paths inside the path.resolve() as the arguments are resolved by processing the arguments or path segments from right to left, with each subsequent path prepended until an absolute path is constructed.

syntax

  • path.resolve will always result in an absolute URL, and will use your working directory as a base to resolve this path

  • After resolving completely if the resulting path doesnâ€™t result in an absolute path, the current working directory is used.

  • The resulting path is normalized and trailing slashes are removed unless the path is resolved to the root directory.

  • Path segments of length zero are ignored.

  • If no path segments are passed as arguments, path.resolve() will return the absolute path of the current working directory.
